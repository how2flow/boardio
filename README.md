# boardio

The python scripts for testing gpio/alts of boards.

### env

```
Linux OS
(developed in Ubuntu 20.04 LTS)
```

### requirements
```
python3 (>= 3.7)
python3-dev
```

### setup

```
$ ./setup.sh
$ udevadm trigger
$ python3 -m pip install -r requirements.txt
```
